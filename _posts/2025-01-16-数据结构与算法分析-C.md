# C 算法学习笔记



## 1. 基础语法



### 1.1 函数传递数组的方法

- 方法一：使用指针

```c
#include <stdio.h>

double getAverage(int * arr, int size){
    int i;
    double sum =0;
    for ( i = 0; i < size; i++)
    {
        sum += arr[i];
    }

    return sum/size;
}

int main(){
    int balance[5] = {1000, 2, 3, 17, 50};
    double avg= getAverage(balance, 5);
    printf("Average value is: %f", avg);
    return 0;
}
```



- 方法二： 使用固定大小的数组

```c
#include <stdio.h>

//传递的是数组的地址，修改数组，会改变原来的数组
//不会检查数组越界问题
double getAverage(int arr[5], int size){
    int i;
    double sum =0;
    for ( i = 0; i < size; i++)
    {
        sum += arr[i];
    }

    return sum/size;
}

int main(){
    int balance[6] = {1000, 2, 3, 17, 50,60};
    //这里传递数组地址
    double avg= getAverage(balance, 6);
    printf("Average value is: %f\n", avg);

    int prices[5] = {100, 200, 300, 400, 500};
    double avg2 = getAverage(prices, 5);
    printf("Average value is: %f\n", avg2);
    return 0;
}
```
